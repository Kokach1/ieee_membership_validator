# 🚀 Quick Start - IEEE Membership Validator Extension

## 3-Step Installation

### Step 1: Open Chrome Extensions
1. Copy this URL: `chrome://extensions/`
2. Paste into Chrome address bar
3. Press Enter

### Step 2: Enable Developer Mode
- Find toggle in **top-right corner** that says "Developer mode"
- Click to enable (turns blue)

### Step 3: Load Extension
1. Click **"Load unpacked"** button (top-left)
2. Navigate to: `c:\Users\KOKACHI\Downloads\projects\claud exp\ieee-validator-extension`
3. Click **"Select Folder"**
4. Extension should appear with IEEE icon ✅

---

## First Use

### 1. Create Excel File
- Column required: `membership_id`
- Optional: `name`, `email`, `dept`

**Example:**
```
| membership_id | name      |
|---------------|-----------|
| 12345678      | John Doe  |
| 87654321      | Jane Smith|
```

### 2. Open IEEE Page
- URL: https://services24.ieee.org/membership-validator.html
- **Log in manually** (important!)

### 3. Use Extension
1. Click extension icon in toolbar
2. Upload your Excel file
3. Click "Start Validation"
4. Wait for completion
5. Click "Export Results"

## ⚠️ Troubleshooting

**Extension doesn't show up?**
- Make sure Developer mode is ON
- Reload the page: `chrome://extensions/`
- Check for errors on extension card

**Can't start validation?**
- Must be on IEEE validator page
- Page must be fully loaded
- Must be logged in

**Need help?**
- Open extension popup
- Right-click → "Inspect"
- Check Console tab for errors
